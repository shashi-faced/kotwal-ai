# Kotwal AI workspace

This repository powers the Kotwal-branded assistant experience. It is a straightforward Vite + React + TypeScript app with Tailwind CSS and shadcn-ui components.

## Getting started locally

1. Install dependencies  
   ```sh
   npm install
   ```
2. Start the dev server  
   ```sh
   npm run dev
   ```

## Tech stack

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn-ui

## Deployment

Build the production bundle with:

```sh
npm run build
```

Serve the `dist/` directory with any static host or your preferred platform (Netlify, Vercel, Cloudflare Pages, etc.).
